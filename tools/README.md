# 工具列表

该目录包含用于 PDF 转图片的实用脚本，适合快速批量处理和单文件转换。

## PDF 工具

| 文件 | 功能说明 |
| --- | --- |
| [pdf_2_png.py](./pdf_2_png.py) | 单个 PDF 文件转换 PNG 图片，或者 PDF 目录批量转换为 PNG 图片 |

## 使用说明

1. 进入当前目录：

   ```bash
   cd ./me-agent/tools
   ```

2. 批量转换：

   ```bash
   python pdf_2_png.py <PDF文件目录>
   ```

3. 单文件转换：

   ```bash
   python pdf_2_png.py <PDF文件路径>
   ```

> 请根据脚本内部说明补充命令参数和依赖环境。
